    Przykładowa animacja:

    Pełna definicja animacji w @keyframes i użycie jej za pomocą shorthandu animation 

    @keyframes makingRounds {
  0% {
    border-radius: 4em;
    border-end-end-radius: 1em;
    translate: calc(-50% + 1em) calc(-50% + 1em) 0;
}

  25% {
    border-radius: 4em;
    border-end-start-radius: 1em;
    translate: calc(50% - 1em) calc(-50% + 1em) 0;
  }

  50% {
    border-radius: 4em;
    border-start-start-radius: 1em;
    translate: calc(50% - 1em) calc(50% - 1em) 0;
  }

  75% {
    border-radius: 4em;
    border-start-end-radius: 1em;
    translate: calc(-50% + 1em) calc(50% - 1em) 0;
  }

  100% {
    border-radius: 4em;
    border-end-end-radius: 1em;
    translate: calc(-50% + 1em) calc(-50% + 1em) 0;
  }
}

div {
  width: 25%;
  aspect-ratio: 1;
  border: var(--border-thick) solid var(--color-sub);
  background-color: var(--color-accent);
  animation: 2s ease-in 0s infinite normal both running makingRounds;
}
