• Element formularza: <form>
  • Etykieta: element <label>
  • Pole wejściowe: element <input>
  • Tekst wielowierszowy: element <textarea>
  • Menu rozwijane: element <select>
  • Element autouzupełniania: <datalist>
  • Grupowanie: <fieldset> i <legend>

    • Element etykiety: <label>

    <label for="<string>"></label>

    Atrybuty

    for
    Łączy etykietę z elementem wejściowym (<input>, <textarea> itd), pozwalając łatwo zidentyfikować, jakiego pola dotyczy. Wymagany, jeśli element wejściowy nie jest potomkiem elementu etykiety.

    Powiązanie etykiety z inputem

    <form>
  <label for="username">Nazwa użytkownika</label>
  <input id="username" />
</form>

    Powiązanie etykiety przez zagnieżdżenie

    Lepiej robić to za pomocą atrybutu for - niektóre technologie asystujące mają problemy z czytaniem inputów zagnieżdżonych w labelach bez for.

    <form action="#" class="example-form" method="post" name="example-form-nested">
  <label> <!-- można uniknąć atrybutu `for` -->
    Nazwa użytkownika <br />
    <input id="username" />
  </label>
</form>
