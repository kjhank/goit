    • Tekst wielowierszowy: element <textarea> 

    <textarea
  autocomplete="off | on"
  autofocus?
  cols="<number>"
  disabled?
  form="<string>"
  maxlength="<number>"
  minlength="<number>"
  name="<string>"
  placeholder="<string>"
  readonly?
  required?
  rows="<number>"
  spellcheck="true | default | false"
  wrap="hard | soft"
>
  Some default value
</textarea>

    Atrybuty

    autocomplete
    Nazwa pola dla przeglądarki, żeby wiedziała, co ma podpowiadać. Lista dostępnych nazw: HTML attribute: autocomplete.
    autofocus
    Określa, czy pole ma automatycznie dostawać focus po załadowaniu
    cols
    Liczba kolumn tekstu
    disabled
    Oznacza pole jako nieaktywne.
    form
    Powiązanie z id elementu form.
    minlength / maxlength
    Odpowiednio minimalna/maksymalna długość (ilość znaków) wartości pola. Raczej unikamy ze względów dostępności.
    name
    Unikalna nazwa pola, konieczna przy wysyłaniu formularza na serwer.
    placeholder
    Wartość zastępcza, pokazywana kiedy input jest pusty.
    readonly
    Blokuje edycję wartości pola.
    required
    Wskazuje, czy pole jest obowiązkowe.
    rows
    Liczba rzędów tekstu
    spellcheck
    Określa, czy zawartość ma być sprawdzana pod względem poprawności językowej
    wrap
    Określa sposób łamania wierszy tekstu

    Formularz z textarea i inputem

    <form action="#" class="example-form" method="post" name="example-form">
  <label for="username">Nazwa użytkownika</label>
  <input type="text" name="username" id="username" placeholder="Tadeusz Kościuszko" required />
  <label for="textarea">Wypracowanie</label>
  <textarea cols="80" name="textarea" id="textarea" placeholder="A long time ago etc." rows="10"></textarea>
  <hr />
  <button type="submit">Wyślij</button>
</form>
