    Atrybuty

    Element select

    autocomplete
    Nazwa pola dla przeglądarki, żeby wiedziała, co ma podpowiadać. Lista dostępnych nazw: HTML attribute: autocomplete.
    autofocus
    Określa, czy pole ma automatycznie dostawać focus po załadowaniu
    disabled
    Oznacza pole jako nieaktywne.
    form
    multiple
    Określa, czy można wybrać więcej niż jedną wartość.
    Powiązanie z id elementu form.
    name
    Unikalna nazwa pola, konieczna przy wysyłaniu formularza na serwer.
    required
    Wskazuje, czy pole jest obowiązkowe.
    size
    Określa rozmiar w liniach tekstu

    Element option

    disabled
    Oznacza pole jako nieaktywne.
    label
    Opisuje wartość
    selected
    Określa, czy element jest domyślnie zaznaczony.
    value
    Wartość
