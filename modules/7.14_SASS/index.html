    SASS - zmienne, zagnieżdżanie i łączenie selektorów

    Deklaracja i użycie zmiennych

    $variableName: value;

selector {
  property: $variableName;
}

    Przykładowe użycie zmiennych

    $saturation: 67%;
$ligthness: 60%;
$mainColor: hsl(90 $saturation $ligthness);
$subColor: hsl(210 $saturation $ligthness);

body {
  background: hsl(0 0% 10%);
}

article {
  color: $mainColor;
}

article > h2 {
  color: $subColor;
}
              
    body {
  background: #1a1a1a;
}

article {
  color: #99dd55;
}

article > h2 {
  color: #5599dd;
}

    Zasięg zmiennych

    Zmiana wartości zmiennej wewnątrz bloku

    $hue: 210;
$mainColor: hsl($hue 70% 65%);

body {
  color: $mainColor;
}

h2 {
  $hue: 90;
}
            
    body {
  color: #67a6e4;
  background: #1a1a1a;
}

h2 {
  color: #a6e467;
}

    Zagnieżdżanie (nesting) reguł

    Przykłady zagnieżdżeń

    a {
  color: limegreen;
}

li {
  list-style-type: disc;
}

.main-header {
  li {
    list-style-type: circle;
    > a {
      color: rebeccapurple;
    }
  }
}
            
    .main-header li {
  list-style-type: circle;
}
.main-header li > a {
  color: rebeccapurple;
}

    Łączenie selektorów

    Przykłady łączenia (konkatenacji) selektorów + BEM

    $mainColor: hsl(210 70% 60%);
$subColor: hsl(90 70% 60%);

.main-footer {
  background-color: hsl(0 0% 5%);

  &__links {
    display: flex;
    gap: 1em;
    align-items: center;
    list-style-type: none;
  }

  &__link {
    color: $mainColor;

    &:hover {
      color: $subColor;
    }

    &--hero {
      scale: 1.05;
    }
  }
}
            
    .main-footer {
  background-color: #0d0d0d;
}
.main-footer__links {
  display: flex;
  gap: 1em;
  align-items: center;
  list-style-type: none;
}
.main-footer__link {
  color: #5299e0;
}
.main-footer__link:hover {
  color: #99e052;
}
.main-footer__link--hero {
  scale: 1.05;
}
