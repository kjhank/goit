    Pliki SVG

    Podstawowa składnia pliku SVG

    <svg xmlns="http://www.w3.org/2000/svg"> <!-- XML namespace attribute is required -->
  <!-- shape definitions -->
</svg>

    Viewport w SVG - atrybuty width oraz height

    <svg
  xmlns="http://www.w3.org/2000/svg">
  width="<width>"
  height="<width>"
>
  <!-- shape definitions -->
</svg>

    Atrybut viewBox

    <svg
  height="<width>"
  viewBox="<min-x min-y width height>"
  width="<width>"
  xmlns="http://www.w3.org/2000/svg">
>
  <!-- shape definitions -->
</svg>

<!-- min-x: min left coordinate -->
<!-- min-y: min top coordinate -->

    Jeśli width i height w atrybucie viewBox są takie same jak atrybuty, odpowiednio, width i height - wektory zostaną wyrenderowane w oryginalnym rozmiarze. W innych przypadkach, zawartość będzie przeskalowana.

    viewBox i skalowanie

    • <svg width="400" height="200" viewBox="0 0 400 200">
  <circle r="75" cx="50%" cy="50%" />
</svg>
              
    • <svg width="400" height="200" viewBox="0 0 200 100">
  <circle r="75" cx="50%" cy="50%" />
</svg>
              
    • <svg width="400" height="200" viewBox="0 0 800 400">
  <circle r="75" cx="50%" cy="50%" />
</svg>
