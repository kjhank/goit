    Elementy tabeli

    table
    Całość tabeli, najbardziej zewnętrzny element
    thead
    Table head: nagłówek tabeli
    tbody
    Table body: treść tabeli
    tr
    Table row: rząd tabeli
    th
    Table header: pojedyncza komórka nagłówka
    td
    Table data cell: Komórka tabeli
    tfoot
    Table footer: Stopka tabeli
    caption
    Opis tabeli
